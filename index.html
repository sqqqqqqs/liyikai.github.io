Custom Game Engine

This is a multi-platform (Mac, Windows, and Linux) game engine made from scratch starting from zero.Built in C++, with Lua scripting support. Besides using a component-based architecture for developers, my engine use SDL for all rendering logics, use Box2D for collision detection, raycasting and physics simulation, use Lua/LuaBridge to easily embed lua codes, GLM to professionalize data types. Besides, there's built-in ParticleSystem for effects, Animator for easy animation and NetworkComponent for multi-player local connections.

Mini-Minecraft

A Minecraft-inspired voxel sandbox built in C++/OpenGL, featuring chunk-based infinite-world streaming, procedural terrain & biome generation driven by multi-noise fields, multithreaded chunk meshing, water rendering, basic player physics & collision, dynamic lighting, and post-processing for improved visual quality. Also includes an in-game crafting system and robust world update logic for efficient real-time interaction.
